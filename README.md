# Project_3_Portfolio_CSS

## Overview
Project_3_Portfolio_CSS is a personal portfolio website built using HTML and CSS. It allows students and professionals to showcase their skills, projects, and contact information in a clean and visually engaging manner[web:12][web:19].

## Features
- Responsive design for desktops and mobiles
- Simple navigation: About, Skills, Projects, Contact
- Easily customizable sections
- Good for recruiters, educators, and collaborators[web:16][web:15]

## Getting Started

### Clone the Repository
git clone https://github.com/shreshthkchaudhary/Project_3_Portfolio_CSS.git

### Usage
- Open `index.html` (or main HTML file) in any web browser
- Edit HTML/CSS files to update your skills, projects, or contact details[web:14][web:18]

- CSS3
